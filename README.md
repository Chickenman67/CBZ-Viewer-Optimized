# 📚 CBZ Reader

A smooth, lightweight CBZ (comic book zip) reader you can run right in your browser.  
Just upload your files and start reading—no installs, no clutter.

---

## 🚀 How It Works

- Open one or more `.cbz` files
- The reader extracts all the pages automatically
- It organizes them in order and gets them ready instantly
- Switch between different reading styles:
  - 📄 Single page
  - 📖 Two-page spread (like a real book)
  - 📜 Scroll mode (binge read)
- Adjust zoom and go fullscreen for a better experience

---

## ⌨️ Keybinds

- → Next page  
- ← Previous page  
- F Toggle fullscreen
